AdSense / Analytics setup (consent-gated)

Dit document laat zien hoe je scripts toevoegt die pas laden na toestemming. Dit helpt bij EU-compliance.

Home Over Voorwaarden Cookie policy Cookie voorkeuren

1. Waar voeg je scripts toe? Aanrader

Plaats optionele scripts in de <head> of net voor </body> op elke pagina. Gebruik data-consent zodat ze pas na toestemming laden.

2. Google Analytics (voorbeeld) analytics

Vervang G-XXXX door je eigen ID.

<script data-consent="analytics" data-src="https://www.googletagmanager.com/gtag/js?id=G-XXXX"></script>
<script data-consent="analytics">
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'G-XXXX', { anonymize_ip: true });
</script>

3. Google AdSense (voorbeeld) ads

Vervang ca-pub-XXXX door je publisher id.

<script data-consent="ads"
  data-src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXX"
  crossorigin="anonymous"></script>

Let op: AdSense implementatie verschilt per site-opzet. Voeg daarna ad units toe volgens AdSense instructies.
